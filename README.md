# 🍎 osFolio — Chirag's Interactive Portfolio

> **A macOS-style desktop environment in the browser** — explore my projects, skills, and experience through an immersive, interactive OS.

[![Live Demo](https://img.shields.io/badge/Live-Demo-6c5ce7?style=for-the-badge&logo=vercel)](https://chirag.dev)
[![GitHub](https://img.shields.io/badge/GitHub-chirag--says-181717?style=for-the-badge&logo=github)](https://github.com/chirag-says/osFolio)

---

## ✨ Features

| Feature | Description |
|---------|------------|
| 🖥️ **Desktop Environment** | Full macOS-style UI with draggable, resizable windows |
| 📂 **Working Apps** | Terminal, VS Code, Camera, Calculator, Calendar, Gallery, and more |
| 💻 **Interactive Terminal** | Type `whoami`, `projects`, `skills`, `neofetch`, and even `sudo rm -rf /` 😏 |
| 📄 **Resume Viewer** | Embedded PDF viewer with download option |
| 🎨 **Theme System** | Light/Dark mode with custom wallpapers |
| 🐣 **Easter Eggs** | Click the Apple logo 5 times... 🎊 |
| 📸 **Live Camera** | Real webcam integration with FaceTime-style UI |
| 🌊 **Glassmorphism** | Premium frosted glass effects on dock & menu bar |
| ⚡ **Boot Animation** | Custom startup screen with personalized branding |

## 🛠️ Tech Stack

- **Framework**: [Svelte 5](https://svelte.dev) + [Vite](https://vitejs.dev)
- **Language**: TypeScript
- **Styling**: Vanilla CSS with CSS variables
- **Icons**: Iconify (Simple Icons + Phosphor)
- **Deployment**: Vercel

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/chirag-says/osFolio.git
cd osFolio

# Install dependencies
pnpm install

# Start the dev server
pnpm dev

# Build for production
pnpm build
```

## 🎯 Terminal Commands

```
whoami          → Who am I?
skills          → Technical skills
projects        → List all projects
experience      → Work experience
education       → Education details
contact         → Contact information
resume          → Download resume
neofetch        → System information
sudo hire chirag → Try it ;)
sudo rm -rf /   → Don't do it! 💀
```
