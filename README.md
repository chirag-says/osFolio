# DevFolio OS

![DevFolio OS Screenshot](public/wallpapers/33.webp)

DevFolio OS is an interactive, macOS-inspired web portfolio built using Svelte. It aims to recreate the desktop experience inside the browser, complete with apps, windows, and desktop management.

### Features
* Fluid UI & Animations mimicking macOS.
* Working Apps (Finder, VSCode, Terminal, Camera, Gallery, etc.)
* Themeing (Light/Dark Mode, Custom Wallpapers)
* Built using modern web technologies: **Svelte** and **Vite**

## Development

```bash
# Install dependencies
npm i

# Start the dev server
npm run dev

# Build for production
npm run build
```
